astro-strip-whitespace

Whitespace Demo

This component intentionally contains extra whitespace between tags and text nodes.

AB

To preserve whitespace between items, you can use {" "} like in JSX/TSX.

Alternatively, you may use an HTML entity like &#32; or &#13; for a space or line break respectively.

Build output should have stripped whitespace.

Note: MD/MDX rendering may include extra whitespace.
